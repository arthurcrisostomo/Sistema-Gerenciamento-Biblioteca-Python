# 📚 Sistema de Biblioteca em Python

Sistema desenvolvido em Python para cadastro e gerenciamento de livros com integração com Excel.

## Senha de acesso ADMIN para adição de livro: 123

## Funcionalidades
- Cadastro de livros
- Listagem de livros
- Busca de livros no banco de dados(Excel)
- Alugar livros
- Devolver livros
- Remover livro do banco de dados
- Integração com Excel

## Tecnologias
- Python
- Excel
